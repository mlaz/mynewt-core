ble\_gap\_adv\_active
---------------------

.. code:: c

    int
    ble_gap_adv_active(void)

Description
~~~~~~~~~~~

Indicates whether an advertisement procedure is currently in progress.

Parameters
~~~~~~~~~~

None

Returned values
~~~~~~~~~~~~~~~

+-----------+-------------------------------------------+
| *Value*   | *Condition*                               |
+===========+===========================================+
| 0         | No advertisement procedure in progress.   |
+-----------+-------------------------------------------+
| 1         | Advertisement procedure in progress.      |
+-----------+-------------------------------------------+
